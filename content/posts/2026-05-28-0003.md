## 2. 迁移操作

### 2.1 关闭 WSL

```bash
wsl --shutdown
```

### 2.2 导出系统镜像到 D 盘

```bash
wsl --export Ubuntu-24.04 D:\workspace\wsl-backup\ubuntu24.tar
```

### 2.3 注销原系统

```bash
wsl --unregister Ubuntu-24.04
```

### 2.4 导入系统到新位置

```bash
wsl --import Ubuntu-24.04 D:\workspace\wslubuntuspace\ubuntu2404 D:\workspace\wsl-backup\ubuntu24.tar --version 2
```

### 2.5 启动新系统

```bash
wsl -d Ubuntu-24.04
```

### 2.6 验证迁移结果

```bash
wsl -l -v
```
